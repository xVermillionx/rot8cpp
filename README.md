# rot8cpp

Just a small rotation to be daemon for auto rotating screen.


## Initial
```bash
cmake .

```

## Build
```bash
make

```

## Install
```bash
sudo make install

```

## Uninstall
```bash
sudo make uninstall

```

## Clean
```bash
make clean

```

## Debug Build
```bash
cmake -DCMAKE_BUILD_TYPE=Debug .

```
